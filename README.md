# Health_Bar - program description
- This is program that you can shout bullets to the rectangle and the rectangle has health bar

# HOW TO USE IT?
- If you press the left click on mouse the triangle will shout bullets
- You can change the bullet damage and rectangle health by using the text boxs on the left
- You change the color of the triangle and the rectangle by using the sliders on the left (red, green, blue)

# HOW TO RUN IT?
- Download python
- run the main file
